## Résumé des emplacements

Pour chaque langue, ajouter ces blocs aux bons emplacements:

1. **vision.json**: Remplacer le bloc `"favorites"` existant (lignes ~166-172) par la nouvelle version complète
2. **common.json**: Ajouter `items`, `items_plural`, `more` après le bloc `"meals"` (vers ligne ~56)

## Vérification

Après avoir ajouté toutes les traductions:

```bash
# Vérifier que tous les fichiers sont valides JSON
cd frontend/src/i18n/locales
for lang in de es pt zh ar; do
  echo "Checking $lang..."
  python -m json.tool $lang/vision.json > /dev/null && echo "  vision.json: OK"
  python -m json.tool $lang/common.json > /dev/null && echo "  common.json: OK"
done
```
